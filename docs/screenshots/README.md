# Screenshots

Hier liegen die Screenshots für die README. So erstellst du sie (Web-Vorschau,
am schnellsten):

1. App im Browser starten:
   ```powershell
   flutter run -d edge --dart-define-from-file=env.json
   ```
2. Anmelden und ein paar Beispiel-Buchungen anlegen (für aussagekräftige Bilder).
3. Browserfenster auf ~390 px Breite ziehen (Handy-Ansicht) bzw. breit lassen
   (Desktop-Ansicht) und je Tab einen Screenshot machen.
4. Bilder mit genau diesen Namen hier ablegen (PNG):

   | Datei                | Inhalt                                   |
   |----------------------|------------------------------------------|
   | `01-konten.png`      | Konten-Tab mit Gesamtvermögen            |
   | `02-buchungen.png`   | Buchungen-Tab (Zeitraum + Summen)        |
   | `03-buchung-neu.png` | Buchung erfassen (mit Tags/Aufteilen)    |
   | `04-statistik.png`   | Statistik (Kategorie-Aufschlüsselung)    |
   | `05-onboarding.png`  | Erststart-Einrichtung (Supabase)         |
   | `06-desktop.png`     | Breite Ansicht (NavigationRail)          |

Danach erscheinen sie automatisch im Abschnitt **Screenshots** der
[Haupt-README](../../README.md).

> Tipp: Auf Windows kannst du mit `Win`+`Shift`+`S` einen Ausschnitt machen.
